# Road Side Grocery - Interactive Landing Page

A vibrant, interactive landing page for a roadside grocery shop showcasing fresh farm produce with modern design and engaging user experience.

## Features

- **Product Showcase** - 12 fresh products across 4 categories (Vegetables, Fruits, Dairy, Bakery)
- **Category Filtering** - Filter products by type with animated buttons
- **Shopping Cart** - Add/remove items with real-time total calculation
- **Product Modals** - Click products for detailed information
- **Testimonial Carousel** - Auto-rotating customer reviews
- **Newsletter Signup** - Email subscription form
- **Responsive Design** - Works on all devices
- **Smooth Animations** - Hover effects, floating elements, and scroll interactions

## File Structure

```
├── index.html    # Main HTML structure
├── style.css     # Styling with animations
└── script.js     # Interactive functionality
```

## Getting Started

1. Open `index.html` in a web browser
2. No build process or dependencies required
3. Fully functional with vanilla HTML, CSS, and JavaScript

## Design Highlights

- **Color Palette**: Earthy greens, warm oranges, and cream tones
- **Typography**: Playfair Display for headings, Lato for body text
- **Animations**: Floating emojis, hover effects, smooth transitions
- **Responsive**: Grid layouts adapt to all screen sizes

## Browser Support

Works on all modern browsers including Chrome, Firefox, Safari, and Edge.
